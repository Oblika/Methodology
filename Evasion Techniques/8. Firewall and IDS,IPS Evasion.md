### 3. Archiving the Payload

Tani, provoni ta arkivoni dy herë, duke i vendosur fjalëkalimin të dy arkivave pas krijimit dhe duke hequr prapashtesën `.rar`/ `.zip`/ `.7z`nga emrat e tyre. Për këtë qëllim, mund të instalojmë [programin RAR](https://www.rarlab.com/download.htm) nga RARLabs, i cili funksionon saktësisht si WinRAR në Windows.

```bash
wget <https://www.rarlab.com/rar/rarlinux-x64-612.ta>
tar -xzvf rarlinux-x64-612.tar.gz && cd rar
rar a ~/test.rar -p ~/test.js
```

Removing the .RAR Extension
```bash
mv test.rar test
ls

test   test.js
```

Archiving the Payload Again
```
rar a test2.rar -p test
```

Removing the .RAR Extension
```
mv test2.rar test2

ls

test   test2   test.js
```
