ICMP tunneling encapsulates your traffic within `ICMP packets` containing `echo requests` and `responses`. ICMP tunneling would only work when ping responses are permitted within a firewalled network.

```bash
# Klonimi i mjetit ptunnel-ng
git clone <https://github.com/utoni/ptunnel-ng.git>

 # Instalimi i mjeteve për ndërtim automatik
sudo apt install automake autoconf -y

# Ndërtimi i ptunnel-ng me binar statik
cd ptunnel-ng/  
sed -i '$s/.*/LDFLAGS=-static "${NEW_WD}\\/configure" --enable-static $@ \\&\\& make clean \\&\\& make -j${BUILDJOBS:-4} all/' autogen.sh  # Modifikon skriptin për ndërtim statik
./autogen.sh  # Ndërton ptunnel-ng

# Transferimi i ptunnel-ng në hostin e pivotit
scp -r ptunnel-ng ubuntu@10.129.202.64:~/

# Nisja e serverit ptunnel-ng në hostin target
sudo ./ptunnel-ng -r10.129.202.64 -R22

# Nisja e klientit ptunnel-ng nga attack host
sudo ./ptunnel-ng -p10.129.202.64 -l2222 -r10.129.202.64 -R22

# SSH përmes tunelit ICMP
ssh -p2222 -lubuntu 127.0.0.1

# SSH me port forwarding dinamik (SOCKS)
ssh -D 9050 -p2222 -lubuntu 127.0.0.1

# Përdorimi i proxychains për skanime
proxychains nmap -sT -Pn -p22 172.16.5.10
```
