## Types of Modules in Metasploit

| **Type**    | **Përshkrimi**                                                      |
| ----------- | ------------------------------------------------------------------- |
| `exploit`   | Shfrytëzon dobësi për të dërguar payload                            |
| `payload`   | Krijon lidhjen me sistemin e komprometuar                           |
| `encoder`   | Kodon payload për të shmangur AV                                    |
| `auxiliary` | Scanning, sniffing, fuzzer, etj.                                    |
| `post`      | Veprime pas shfrytëzimit (info gathering, pivoting, hashdump, etj.) |
| `nop`       | No Operation (ruan madhësinë e payload-it)                          |
| `plugin`    | Shton skripta shtesë në msfconsole                                  |
