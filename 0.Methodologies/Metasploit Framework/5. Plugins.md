
Plugin-et jane programe te gatshme te bera nga persona te tjere qe mund ti integrojm ne Metasploit.

Te gjitha pluginet e Metasploit ruhen ne kete direktorit

```bash
 ls /usr/share/metasploit-framework/plugins
```

```bash
# Mund te shkarkojm plugin te gatshem
git clone <https://github.com/darkoperator/Metasploit-Plugins>

# Me pas mund ti kopjojm ne folderin e Metasploit
sudo cp ./Metasploit-Plugins/pentest.rb /usr/share/metasploit-framework/plugins/pentest.rb

# Pastaj mund ta testojm duke hapur msfconsole
load pentest
```
