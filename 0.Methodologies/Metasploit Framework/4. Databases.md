
1. Database në Metasploit përdoret për të ruajtur rezultatet e skanimeve, hostet, shërbimet, kredencialet dhe loot.

Përdor PostgreSQL si backend dhe lejon integrimin e mjeteve si Nmap brenda MSF.
```bash
# =====================================================
# PostgreSQL Service
# =====================================================

# Kontrollojm statusin e PostgreSQL
sudo service postgresql status

# Nisim PostgreSQL
sudo systemctl start postgresql

# =====================================================
# Inicializimi i MSF Database
# =====================================================

# Krijojm dhe inicializojm databazen e Metasploit
sudo msfdb init

# Kontrollojm statusin e DB
sudo msfdb status

# Nisim msfconsole te lidhur me DB
sudo msfdb run
```

**Nëse DB ka probleme**
```bash
# Re-inicializojm databazen
msfdb reinit

# Kopjojm file konfigurimi manualisht
cp /usr/share/metasploit-framework/config/database.yml ~/.msf4/

# Restartojm PostgreSQL
sudo service postgresql restart

# Nisim msfconsole pa output shtesë
msfconsole -q
```

MSF - Database Options
```bash
# Shfaq te gjitha komandat e database
msf6 > help database

# Kontrollojm nqs DB eshte e lidhur
msf6 > db_status
```

Workspaces
```bash
# Shfaq workspaces ekzistuese
msf6 > workspace

# Krijojm workspace te ri
msf6 > workspace -a Target_1

# Nderrojm workspace
msf6 > workspace Target_1
```

Importing Scan Results
```bash
# Importojm rezultate skanimi (p.sh. Nmap XML)
msf6 > db_import Target.xml
```

Using Nmap Inside MSFconsole
```bash
# Skanim me Nmap nga brenda Metasploit
msf6 > db_nmap -sV -sS 10.10.10.8
```

Data Backup
```bash
# Eksportojm databazen ne file XML
msf6 > db_export -f xml backup.xml
```

Credentials
```bash
# Shfaq opsionet e kredencialeve
msf6 > creds -h

# Shfaq kredencialet e ruajtura
msf6 > creds
```

Loot
```bash
# Shfaq opsionet e loot
msf6 > loot -h

# Shfaq te gjitha te dhenat e mbledhura
msf6 > loot
```
