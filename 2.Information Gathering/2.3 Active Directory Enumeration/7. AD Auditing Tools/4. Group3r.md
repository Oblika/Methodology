
[Group3r](https://github.com/Group3r/Group3r) është një mjet që analizon **Group Policy Objects (GPO)** në një domain Active Directory dhe gjen:

- konfigurime të dobëta
- politika të rrezikshme
- keq-konfigurime që ndikojnë sigurinë
- vendosje të gabuara të regjistrit
- leje të tepërta
- opcione që hapin rrugë për lateral movement / priv esc

# **Si funksionon?**

Duhet të jesh:

- në një host të domain-joined
- i loguar si një **domain user** (s’ka nevojë për admin)

```bash
group3r.exe -s

# ose ta ruajm ne nje file

group3r.exe -f gpo_results.log
```
