
[ADRecon](https://github.com/adrecon/ADRecon) është një **mjet PowerShell i krijuar për të mbledhur automatikisht të gjitha informacionet kritike të Active Directory** dhe për t’i paraqitur ato në **raporte HTML dhe CSV** të strukturuara bukur.

✔ mbledhësh _gjithçka_ nga AD në pak minuta  
✔ analizosh misconfigurations që mund të jenë të humbura me manual enumeration  
✔ krijosh raporte të qarta për klientin


```
.\ADRecon.ps1
```
