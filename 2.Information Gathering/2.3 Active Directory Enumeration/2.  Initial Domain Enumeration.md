
## - Ientify Hosts 


### 🔍 _Key Enumeration Targets_

- Active Directory (AD) Users
- AD-joined Machines
- Domain Controllers & Key Services
- Exposed/Vulnerable Hosts

### 🔎 _Host Discovery_

- **Wireshark** / **TCPDump** (Passive sniffing)
- **Responder** (LLMNR/NBTNS spoofing)
- **FPing** for active ping sweep:

```bash
# Kapim trafikun dhe shikojm protokollet ARP per te zbuluar hostet qe jan aktive
sudo -E wireshark

# Mund te perdorim dhe tcpdump
sudo tcpdump -i ens224

# Po ashtu dhe responder per te kapur trafikun
sudo responder -I ens224 -A

# fping do na tregoj te gjitha IP qe jan aktive ne rrjete
fping -asgq 172.16.5.0/23
```

##  - Identify Users

User Enumeration using Kerbrute with BruteForce
```bash
sudo git clone <https://github.com/ropnop/kerbrute.git>
make help
sudo make all
./kerbrute_linux_amd64 
sudo mv kerbrute_linux_amd64 /usr/local/bin/kerbrute

kerbrute userenum -d INLANEFREIGHT.LOCAL --dc 172.16.5.5 jsmith.txt -o valid_ad_users
```

## - Identify Password Policy 


````bash
**# ----------------- from Linux -------------------------**

# Can we see the password policy from Linux
crackmapexec smb 172.16.5.5 -u avazquez -p Password123 --pass-pol

# We can connect to rpcclient and view password policies
rpcclient -U "" -N 172.16.5.5
> querydominfo
> getdompwinfo

# We can view password Policies with enum4linux
enum4linux -P 172.16.5.5

# We use enum4linux-ng to take password POlicies and save to a file
enum4linux-ng -P 172.16.5.5 -oA ilfreight

# LDAP Anonymous Bind - Using ldapsearch
ldapsearch -h 172.16.5.5 -x -b "DC=INLANEFREIGHT,DC=LOCAL" -s sub "*" | grep -m 1 -B 10 pwdHistoryLength

**# ----------------- from Windows -------------------------**

# Create a null session
net use \\DC01\ipc$ "" /u:""
# Using net.exe - We can connect to rpcclient and view password policies
net accounts

# Using PowerView
import-module .\PowerView.ps1
Get-DomainPolicy
```
