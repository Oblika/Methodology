


### From  Linux
```bash
# Can we see the password policy from Linux
crackmapexec smb 172.16.5.5 -u avazquez -p Password123 --pass-pol

# We can connect to rpcclient and view password policies
rpcclient -U "" -N 172.16.5.5
> querydominfo
> getdompwinfo

# We can view password Policies with enum4linux
enum4linux -P 172.16.5.5

# We use enum4linux-ng to take password POlicies and save to a file
enum4linux-ng -P 172.16.5.5 -oA ilfreight

# **LDAP Anonymous Bind -** Using ldapsearch
ldapsearch -h 172.16.5.5 -x -b "DC=INLANEFREIGHT,DC=LOCAL" -s sub "*" | grep -m 1 
-B 10 pwdHistoryLength
```
