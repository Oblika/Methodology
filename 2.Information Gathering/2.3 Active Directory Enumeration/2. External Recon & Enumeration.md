

### 🎯 _What Are We Looking For?_

- IP Space
- Domain Info (Names, Subdomains)
- Technology Stack / Schema Format
- Public Data / File Disclosures
- Breach Data (Leaks, Passwords)

### 🌐 _Where to Look_

- **IP & ASN Registrars:**
    - IANA, ARIN, RIPE
    - BGP Toolkit
    - Hurricane Electric
- **Domain & DNS:**
    - DomainTools
    - ICANN
    - PTRArchive
    - [Viewdns.info](http://Viewdns.info)
    - `nslookup <IP>`
- **Public Repositories & Buckets:**
    - GitHub (use dorks, trufflehog)
    - AWS S3, Azure Blob Storage
    - Google Dorks (e.g. `site:s3.amazonaws.com <company>`)
    - Greyhat Warfare
- **Company Website & Job Postings:**
    - Look for email formats, services, file leaks
    - SharePoint Admin / IT job listings
- **Breach Data:**
    - [HaveIBeenPwned](https://haveibeenpwned.com/)
    - DeHashed
- **Social Media:**
    - LinkedIn (for employee names)
    - Use `linkedin2username` for enum
