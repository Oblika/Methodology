
CrackMapExec
```bash
# CME - Domain User Enumeration
sudo crackmapexec smb 172.16.5.5 -u <username> -p <password> --users

# CME - Domain Group Enumeration
sudo crackmapexec smb 172.16.5.5 -u <username> -p <password> --groups

# CME - Logged On Users
sudo crackmapexec smb 172.16.5.130 -u <username> -p <password> --loggedon-users

# Share Enumeration - Domain Controller
sudo crackmapexec smb 172.16.5.5 -u <username> -p <password> --shares

# Spider_plus
sudo crackmapexec smb 172.16.5.5 -u <username> -p <password> -M spider_plus --share 'Share Name'
```

SMBMap
```bash
# SMBMap To Check Access
smbmap -u <username> -p <password> -d INLANEFREIGHT.LOCAL -H 172.16.5.5

# Recursive List Of All Directories
smbmap -u <username> -p <password> -d INLANEFREIGHT.LOCAL -H 172.16.5.5 -R 'Department Shares' --dir-only
```

rpcclient
```bash
# SMB NULL Session with rpcclient
rpcclient -U "" -N 172.16.5.5

# RPCClient User Enumeration By RID
queryuser <RID>

# Enumerate Users and RID
enumdomusers
```

**Impacket Toolkit**
```bash
# Using psexec.py to connect to host with interactive shell
psexec.py inlanefreight.local/<username>:'password'@172.16.5.125  

# Using wmiexec.py to connect to host with a shell not full interactive,a little more hidden
wmiexec.py inlanefreight.local/<username>:'password'@172.16.5.5  
```

**Windapsearch**
```bash
# Windapsearch - Domain Admins
python3 windapsearch.py --dc-ip 172.16.5.5 -u <username>@inlanefreight.local -p <password> --da

# Windapsearch - Privileged Users
python3 windapsearch.py --dc-ip 172.16.5.5 -u <username>@inlanefreight.local -p <password> -PU
```
