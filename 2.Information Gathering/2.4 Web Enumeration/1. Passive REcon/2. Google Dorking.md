
```bash
# Finding Login Pages
site:example.com inurl:login OR inurl:admin

# Identifying Exposed Files
site:example.com filetype:pdf (filetype:pdf OR filetype:docx)

# Uncovering Configuration Files
site:example.com inurl:config.php (ext:conf OR ext:cnf)

# Locating Database Backups
site:example.com inurl:backup
site:example.com filetype:sql
```


|Operator|Example|Description|
|---|---|---|
|`site:`|`site:example.com`|Results only from specified domain|
|`inurl:`|`inurl:admin`|URLs that contain the keyword|
|`intitle:`|`intitle:login`|Titles that include the keyword|
|`filetype:`|`filetype:pdf`|Search for specific file types|
|`ext:`|`ext:sql`|Alternative to filetype|
|`cache:`|`cache:example.com`|View cached version of a page|
|`link:`|`link:example.com`|Find pages linking to a site|
|`related:`|`related:example.com`|Find related websites|
|`allintext:`|`allintext:username password`|Text must appear in body|
|`allintitle:`|`allintitle:login page`|All words in title|
