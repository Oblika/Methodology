
Installing Go, Python and PIPX
```bash
# Update Package
sudo apt update

# Installing go
sudo apt install -y golang

# Install Python
sudo apt install -y python3 python3-pip

# Install and configure pipx
sudo apt install pipx
pipx ensurepath
sudo pipx ensurepath --global

# To ensure that Go and Python are installed correctly, you can check their versions:
go version
python3 --version
```

Installing Tools
```bash
# Install Ffuf
go install github.com/ffuf/ffuf/v2@latest

# Gobuster
go install github.com/OJ/gobuster/v3@latest

# FeroxBuster
curl -sL https://raw.githubusercontent.com/epi052/feroxbuster/main/install-nix.sh | sudo bash -s $HOME/.local/bin

# wfuzz/wenum
pipx install git+https://github.com/WebFuzzForge/wenum
pipx runpip wenum install setuptools
```
