Two phenomenal tools that every tester should have in their arsenal are [EyeWitness](https://github.com/FortyNorthSecurity/EyeWitness) and [Aquatone](https://github.com/michenriksen/aquatone).


### 2. Application Discovery & Enumeration

```bash
# 1) Basic TCP web-port scan (parashikon një file "scope_list" me host: ip ose hostnames, një per rresht)
nmap -p 80,443,8000,8080,8180,8888,10000 --open -oA web_discovery -iL scope_list

# shpjegim: -oA web_discovery -> krijon web_discovery.nmap / .xml / .gnmap
# -iL scope_list -> lexon hostet nga file
# --open -> vetëm hostet me porta të hapura (filtron outputin)

# 2) Single-host service/version probe (kur dëshiron më tepër detaje)
sudo nmap --open -sV 10.129.201.50
# shpjegim: -sV = service/version detection. Sudo vetëm nëse kërkohet (synack kernel raw sockets).

# 3) EyeWitness - screenshots & quick triage
sudo apt update && sudo apt install eyewitness
# Run:
eyewitness --web -x web_discovery.xml -d inlanefreight_eyewitness
# shpjegim: --web për web screenshots; -x input xml nga nmap; -d output dir

# 4) Aquatone - kattis visual triage, parsing Nmap XML
# (download + unzip)
wget <https://github.com/michenriksen/aquatone/releases/download/v1.7.0/aquatone_linux_amd64_1.7.0.zip>
unzip aquatone_linux_amd64_1.7.0.zip
# siguro që binaire të jetë executable dhe në PATH, p.sh:
chmod +x aquatone
mv aquatone /usr/local/bin/

# Run aquatone using the nmap XML
cat web_discovery.xml | aquatone -nmap
# shpjegim: -nmap -> lexon xml nga nmap dhe krijon screenshots + html report.

```
