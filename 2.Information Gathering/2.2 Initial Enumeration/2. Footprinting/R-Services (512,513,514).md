
**R‑services** janë një grup shërbimesh shumë të vjetra të Unix-it që lejojnë _remote login_, _remote shell_ dhe _info për përdoruesit_ **pa fjalëkalim**, vetëm duke besuar IP‑në ose hostname‑in e klientit.

```bash
# Creates a remote login session via rlogi (513)
rlogin <IP> -l <username>

# Let's see which user we're logged in as.
rwho

# Shows active users on a large host
rusers -al <IP>

# Remote Command Execution (512)
rexec -l username -p password 10.10.10.5 id

# Get a interactive Shell (512)
rexec -l username -p password 10.10.10.5 /bin/bash
```

**Përdorimi:**

- Remote login (si SSH i vjetër, por pa enkriptim)
- Enumerim i përdoruesve të kyçur
- Zbulim i përdoruesve aktivë në sistem
