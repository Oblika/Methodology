## smbclient
```bash
# List available SMB shares anonymously
smbclient -N -L //<IP>

# Connect to a specific partition (will ask for credentials if it is protected).
smbclient //<IP>/sharename

smbclient -L //10.129.19.42 -U ''

# Connect to a specific SMB share
smbclient //<FQDN IP>/<share>

# Hyr brenda share-it
smbclient -N //10.129.228.253/Public

# Pasi kemi hyre brenda SMB mund te shkarkojm te gjitha file me kte skript
prompt off  
recurse on  
mget *
```


## smbmap
```bash
# Enumerate shares using smbmap
smbmap -H <IP>

# Recursively list contents of a specific shar
smbmap -H <IP> -r notes

# Download a file from a share
smbmap -H <IP> --download "notes\note.txt"

# Upload a file to a share
smbmap -H <IP> --upload test.txt "notes\test.txt"
```

## rpcclient

# Query SMB services and enumerate users anonymously
